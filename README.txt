Cette page à été réaliser dans le cadre du Hackaton 2024 Organisé par le lycée Chevrollier dans la section BTS SIO.

Elle permet l'utilisitation et la visualisation des données présentes sur l'openData d'Angers Loire Métropole via une API.

Cette page suffit d'être installé sur le reste du site Web de l'entreprise fictive "Parent à tout prix" pour s'intégrer. Elle n'a ni besoin d'un accès à une base de données, ni de préparation au préalable sur le support (siteWeb si il en existait déjà un.)
